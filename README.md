### Recommended

- PHP 8.0 +
- Laravel 9.0 +  
### Installation

**Composer**

```shell
composer require islomiddin-dev/generating-repository-and-service
```

**Example Command**

```
php artisan make:repository User
```

```
php artisan make:service User
```
